大学里很多学生挂科前其实是有信号的——比如最近登录学习平台少了、习题正确率下降了、某个知识点一直没掌握——但传统的学业预警往往要等到期末出成绩才"亮红灯",这时候已经太晚了。本文想做的事很简单:把学生平时在校园系统里留下的"学习痕迹"收集起来,用机器学习的方法提前判断谁有挂科风险,让辅导员能早点介入辅导。系统的核心创新点是把"知识点之间的关联关系"也考虑进去——比如"贝叶斯定理"没掌握,大概率"朴素贝叶斯分类器"也学不好——这种关联不是单独看行为数据能得到的,所以我引入了知识图谱来辅助判断。实验结果显示,这种方法比传统的"光看成绩"和"光看行为"都更准,能提前 28 天预警,准确率达到了 0.87。
