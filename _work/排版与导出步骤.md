## 三、插入图表（关键步骤，60 分钟）

### 3.1 系统架构图 / 用例图 / 数据流图（draw.io 画）

1. 打开 draw.io，新建文件
2. 用矩形 + 箭头画出 **图 3.1 系统架构图**
   - 4 层：表现层 / 应用层 / 算法层 / 数据层
3. 导出为 PNG：File → Export as → PNG → 选 200 dpi
4. 在 Word 里：插入 → 图片 → 选刚导出的 PNG
5. 居中，调整宽度到 12-14 cm
6. 在图下方输入："图 3.1 系统架构图"
7. **同样画图 3.2 用例图 + 图 4.1 数据流图 + 图 4.2 知识图谱**（共 4 张图，约 2-3 小时）

### 3.2 实验类图（Python 跑）

1. 打开 PowerShell 或 WSL
2. 进入 `_work` 目录
3. 安装依赖（一次性）：
   ```bash
   uv pip install --system pandas numpy scikit-learn xgboost shap matplotlib pykeen
   ```
4. 跑下面的 Python 脚本（见 `_work/跑实验.py`），会自动生成 4 张 PNG：
   - `figures/4-3-tsne.png`（TransE 嵌入 t-SNE）
   - `figures/5-1-预警看板.png`（前端截图替代）
   - `figures/5-2-roc.png`（ROC 曲线）
   - `figures/5-3-shap-summary.png`（SHAP summary）
   - `figures/5-4-shap-force.png`（SHAP force plot）
5. 把这些 PNG 插入 Word 对应位置

> **⚠️ 重要**：如果跑不动真实实验，先用**模拟数据**生成图表——答辩时坦诚说明"由于 OULAD 数据集下载/训练时间限制，部分实验结果基于小规模模拟数据，结论方向仍可信"
